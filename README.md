# AI-Assisted Reporting Tool

**Description:**  
A Python and SQL Server-based tool that automates report generation from structured databases. Designed to simplify the creation of trip reports, movement summaries, and offline logs.

**Technologies Used:**  
- Python  
- SQL Server  

**Features:**  
- Automates report generation from database tables  
- Generates summaries and offline logs efficiently  
- Can be extended for additional reporting needs  

**Setup Instructions:**

   ```bash
